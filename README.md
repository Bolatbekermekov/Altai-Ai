# Altai Ai — разработка под ключ

Лендинг нашей команды: веб-приложения, мобильные приложения, Telegram-боты и AI-решения для бизнеса. Быстрый контакт через WhatsApp и понятный процесс работы.

Next.js 15 · Tailwind CSS 4.1 · TypeScript · shadcn/ui

## Чем занимаемся
- Веб-продукты: CRM, панели, маркетплейсы, корпоративные сайты, лендинги.
- Мобильные приложения: iOS/Android (React Native), push, онлайн-оплаты, доставка.
- Telegram-боты: продажи и поддержка, интеграции с CRM/API, платежи, админ-панели.
- AI и автоматизация: чат-боты с NLP, RAG, аналитика, интеграции в процессы.

## Структура лендинга
- Hero: оффер + кнопка в WhatsApp.
- Услуги: e-commerce, лендинги, мобильные, CRM/SaaS, чат-боты, AI.
- Кейсы: Zeep Coffee (мобильное + CRM), StrategicLaw Group (многофункциональный лендинг), Telegram-бот решения.
- Процесс: анализ → разработка → поддержка.
- Отзывы: клиенты из Казахстана (StrategicLaw Group, Zeep Coffee, Frito и др.).
- Прайс: тарифы «Бесплатно», «Pro», «Pro для команды».
- Контакты: форма с авто-текстом в WhatsApp.
